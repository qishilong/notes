# 搭建环境

如果是使用 React Native 构建一个新应用，官方推荐使用框架脚手架工具 [Expo CLI](https://docs.expo.dev/get-started/installation/) 来搭建开发环境。Expo CLI 提供了一套完整的工具链，简化了 React Native 应用的创建和管理过程。

> Expo 是一个生产级 React Native 框架。Expo 提供了一系列开发者工具，使应用程序开发更加便捷，例如基于文件的路由、标准原生模块库等等。

要创建一个新的 Expo 项目，可以在终端中运行以下命令：

```bash
npx create-expo-app@latest
```

创建完成之后，查看 Expo 入门指南的其余部分，开始开发应用。
